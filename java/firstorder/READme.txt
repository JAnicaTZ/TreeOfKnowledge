TreeOfKnowledge – First-Order Logic Calculator (Java, 2002–2025)
===============================================================

Author: Ana Kovačević (JAnica Tesla Zrinski)
Core idea: "Stablo spoznaje" – original tree-based logic calculator from 2002,
adapted for modern Java and web integration.

What this archive contains
--------------------------
- ORIGINAL Java source files for:
  * First-order logic calculator (formulas, parser, tree builder)
  * Propositional logic minimization (in progress / coming soon)
  * Supporting classes (no web frontend, no JavaScript, no HTML)

What this archive does NOT contain
----------------------------------
- No web UI (HTML/CSS/JS)
- No build scripts for deployment on TreeOfKnowledge.eu
- No third-party frameworks – pure Java only.

Requirements
------------
- Java JDK 17 or newer (LTS preferred)
- Maven (optional, if you want to build as a Maven project)

Basic usage (from source)
-------------------------
1. Open the project in your favourite IDE (IntelliJ, VS Code, Eclipse…)
2. Make sure the JDK is correctly configured (Java 17+).
3. Compile all sources.
4. Run the main class that launches the Swing GUI calculator
   (e.g. `CalcLauncher` or dedicated main class in your setup).
